# Passport Local Authentication

## Packages

* bcrypt
* body-parser
* cors
* express
* express-session
* mongoose
* nodemon
* passport
* passport-local

## Quick Start

`npm cache verify`

`npm cache clean --force (Optional)`

`npm install`

`npm start`

## Endpoints 

* Sign Up page :  `http://localhost:5000/api/signup`
* Login Page : `http://localhost:5000/api/login`
* Logout : `http://localhost:5000/api/logout`
